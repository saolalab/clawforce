# Security

## Reporting a Vulnerability

**Do not open a public GitHub issue.** Create a [private security advisory](https://github.com/saolalab/clawforce/security/advisories/new) on GitHub or email the maintainers.

For detailed security documentation (isolation, secrets, tool security, etc.), see the [Security guide](https://saolalab.github.io/clawforce/guide/security) in the docs.
